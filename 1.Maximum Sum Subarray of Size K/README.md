# Problem Statement:  
Given an array of positive integers and a positive number k, the task is to find the maximum sum of any contiguous subarray of size k.

Input:  
array = [3, 5, 2, 1, 7]  
k = 2  

Output:  
8  

Explanation:  
The subarrays of size k=2 are:  

[3, 5] → Sum = 8  
[5, 2] → Sum = 7  
[2, 1] → Sum = 3  
[1, 7] → Sum = 8  
The maximum sum is 8.
